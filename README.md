## 📖 概要

PC、インターネット、モバイル、クラウド。テクノロジーは常に若者が使いこなし、イノベーションを起こしてきた。シニア世代は過去の成功体験やバイアスに囚われ、新しいテクノロジーを受け入れることができなかった。

生成AIで、その構造が初めて逆転した。

AIの出力品質は、使う人間の経験・知識・文脈の深さに依存する。技術史上初めて、経験の蓄積がテクノロジーの優位性に直結する特異点が発生した。

しかし、最も恩恵を受けているシニア世代は、その恩恵の構造を次世代に渡していない。

人類は経験ある世代から次世代へ学びを提供することで進化してきた。AI時代にその連鎖が途絶えようとしていることに対する私たちの文明論的な提言。

本書は、若者に「武装」を促し、シニアに「責務」を伝える。<br>
そして両者の間に、AI時代の新しい世代間契約を提案する。

## 🔑 本書の核心

### 人とAIの共創黄金比「10:80:10の法則」 — 精神的OS

私が提唱している、人とAIの共創黄金比「10:80:10の法則」を、「AI時代に人間が人間であり続けるための精神的OS」として再定義する。

| フェーズ   | 従来の定義   | 本書の再定義                          |
| ------ | ------- | ------------------------------- |
| 最初の10% | 課題設定    | **自分の意志** — 何を問うか、何を求めるか。自分の軸   |
| 80%    | AIによる処理 | AIのアウトプット — ここは任せていい            |
| 最後の10% | 成果物の仕上げ | **判断** — AIの回答が自分の軸に対して適切かを評価する |

### クリティカルシンキング — 自分を守る技術

効率化のためのスキルではない。AIの出力に対して「自分の軸」を保ち続けるための、防衛技術。

### 理論⇔実践の往復運動

理論は本書で学べる。しかし理論だけでは能力にならない。実践の場で理論をぶつけ、その経験を理論に戻す。この往復運動だけが、本質的な能力を形成する。
